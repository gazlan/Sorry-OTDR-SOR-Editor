Заменить таблицу ивентов

Замена (физическая) таблицы ивентов производится на вкладке File info.

После выбора строки с именем блока 'KeyEvents', по правому клику появится контекстное меню. Выбрать в нем пункт 'Replace'.





После этого в появившемся диалоге открытия файла необходимо выбрать SOR-файл из которого будет импортирована таблица ивентов для замены текущей таблицы.





В соответствии с идеологией редактора, никакие изменения не сохраняются в файле, пока не нажата кнопка 'Write'. Нажмите ее.

Рекомендуется не перезаписывать текущий файл, а сохранить изменения под новым именем.
