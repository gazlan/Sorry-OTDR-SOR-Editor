Создать таблицу ивентов

Для создания таблицы ивентов необходимо добавить произвольный ивент (позднее, он может быть удален). Для этого на вкладке 'Review' или 'Slide' выбрать точку трассы, содержащую ивент и по правому клику перейти в контекстное меню.

В этом меню выбрать пункт 'Insert Event'.





После этого откроется окно 2D-редактора с выбранным участком трассы. Редактировать в данный момент ничего не требуется, нажмите 'Save' и закройте диалог.





В соответствии с идеологией редактора, никакие изменения не сохраняются в файле, пока не нажата кнопка 'Write'. Нажмите ее.





Рекомендуется не перезаписывать текущий файл, а сохранить изменения под новым именем.
