SOR Editor support page


Freeware

SOR Editor - клон программы Sorry!, копирующий ее интерфейс и функциональность.

Результатом работы SorEditor является зашифрованный файл с расширением .ENC

Предлагаемые конвертеры ENC-2-TXT и ENC-2-SOR предназначены для дешифровки .ENC-файлов (ENC-2-TXT) созданных программой SorEditor v.3.0 и их конвертации в стандартные .SOR-файлы (ENC-2-SOR).

Конвертер TXT-2-LST предназначен для получения форматированного листинга текстового файла из результата работы конвертера ENC-2-TXT.

Для распаковки архивов требуется программа WinRar версии не ниже 5.0
